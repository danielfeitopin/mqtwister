# Support

Thank you for using MQTwister! If you need help or have questions, please refer to the following options:

## Documentation

- Check the [README](README.md) file for setup and usage instructions.
- Refer to the [SECURITY](SECURITY.md) file for reporting vulnerabilities.

## Community

- Engage with the community by participating in discussions on the [GitHub Discussions page](https://github.com/danielfeitopin/mqtwister/discussions).

## Resources

- **Ettercap**: A comprehensive suite for man-in-the-middle attacks on LAN. Learn more at [Ettercap's official website](https://www.ettercap-project.org/).
- **Scapy MQTT**: Scapy is a powerful Python-based interactive packet manipulation program. The MQTT module allows working with MQTT packets. Check out [Scapy's documentation](https://scapy.readthedocs.io/en/latest/).

We appreciate your feedback and contributions to improve MQTwister!
